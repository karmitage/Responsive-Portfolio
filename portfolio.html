Giphy API

Using the Giphy API, this app allows the user to enter a mood and get related gifs.

Train Scheduler

Create new train schedules & see when the next trains are coming. This project makes use of Firebase to maintain a persistent database of trains.

Hiker

Hiker allows users to search for hikes within a specified radius of any location in the US or Canada. Bonus: You can choose the difficulty level, and even get directions via Google Maps. This project uses the Hiking Project & Google Maps APIs, and Bootstrap for front-end layout & styling.

Liri Node App

The Liri Node App takes command line arguments and displays information about movies, upcoming concerts, and songs. The app makes use of three APIs (Spotify, Bands in Town & OMDB).

Friend Finder

Friend Finder uses Node & Express to create an application that surveys users, stores their responses, and matches them with compatible friends. The application is deployed on Heroku.
