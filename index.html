About Me

I have an undergrad degree in philosophy, and I never envisioned being a programmer. But when I started learning SQL and Python a few years ago, I was hooked. Programming is a lot like math or formal logic, or learning a new language. I love the way coding requires me to solve problems rigorously and creatively.

In my current work as a business analyst, I create ETL processes, build data architecture, & write code in a Hadoop ecosystem. Since enrolling in the UW's full stack web development course, I've learned Javascript, HTML & CSS, & Node. I'm excited to transition my career to software development, and I'm especially looking forward to learning more about data structures & functional programming.

Outside of coding, I'm a backpacker, runner, and (very) amateur ukelele player.
