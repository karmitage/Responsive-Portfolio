About Me

My name is Katherine, and I'm learning full stack web deveopment.

I have an undergrad degree in philosophy, and I never envisioned being a programmer. But when I started learning SQL and Python a few years ago, I was hooked. Programming is a lot like math or formal logic, or learning a new language. I love the way programming requires both rigorous problem-solving and creativity.

In my current work as a business analyst, I write SQL and Hive queries on a daily basis. Since enrolling in the UW's full stack web development course, I've learned Javascript, HTML & CSS, & Node. I'm looking forward to learning more about data structures & functional programming.

Outside of coding, I'm an avid backpacker, runner, and (very) amateur ukelele player.
